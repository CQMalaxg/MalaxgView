# 导出解译结果

GeoView提供两种导出解译结果的方式：下载图像或压缩包。

## 下载结果图

可通过点击结果图下方的下载图标，将单张结果图下载到本地。下载结果将被保存为.png格式。

## 下载压缩包

可点击“结果图打包”将本次解译输出的所有结果图压缩并打包为zip文件。
